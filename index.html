Pomodoro by antigluk

Программа для замера помидор по 25 минут

Программа позволяет вам эффективно управлять своим временем с помощью, так называемой, помидорной техники (pomodoro technique)
Подробнее: http://en.wikipedia.org/wiki/Pomodoro_Technique
На русском: http://tim.com.ua/2009/10/pomodoro-technique/

Dependencies

Python 2.7, wxWidgets, wxPython

Install

Запускать $./pomodoro.py
